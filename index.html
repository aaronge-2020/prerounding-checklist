      1. Open the app in Safari.

      Android tablet

      1. Open the app in Chrome.
      2. Tap Install App if shown, or Add to Home screen.
      3. Use Download HTML for a standalone offline file.

      First-use offline de-ID is not guaranteed because the model must be fetched once. After the first online load, browser caching usually helps; the downloaded HTML remains the safest offline checklist fallback.
